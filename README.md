# RazorFixer
Patches Razor1911's Max Payne 3 cracks and allow multiplayer lan with mods 

# Warning 
this mod is meant to be used with razor's exe (MD5: 22ef59e5943cb1face20ce4c0d72f5b9).
How to install? simply drag the zip in the game folder and select an user name and a unique id inside the INI file
